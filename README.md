Poseidon Software - Barcoded Warehouse & Stock Tracking System
This project is a professional warehouse management solution, fully modernized from a simple prototype on both Frontend and Backend.

🚀 Evolution of the Project
The initial version (Prototype) utilized standard Windows Forms components and a basic code structure. The current version has been completely restructured.

✨ Frontend (UI & UX) Updates
The visual identity has been completely transformed to enhance user experience:

Standard -> Modern: Replaced classic Windows Forms with Guna2 UI for a modern touch.

Dark Mode: Implemented an aesthetic, eye-friendly dark theme design.

Neon Effects: Added neon glows for active components and shadows, matching the brand colors.

Alignment & Layout: All controls are pixel-perfectly aligned within panels using mathematical centering.

🏗️ Backend (Infrastructure) Enhancements
Technical debt was cleared, making the codebase sustainable:

Layered Architecture: Transitioned from "spaghetti code" to structured layers: Data Access (DAL), Business Logic (BLL), and Presentation (UI).

Security: Passwords are now stored using modern hashing algorithms instead of plain text.

SQLite Optimization: Improved database connection management and added auto-generation for missing database files.

Error Handling: Integrated detailed visual error notifications for a better user flow.

This project serves as an example of how a simple prototype can evolve into a professional-grade product.

# Poseidon Yazılım - Barkodlu Depo & Stok Takip Sistemi

Bu proje, basit bir prototipten alınarak **Frontend** ve **Backend** tarafında tamamen modernize edilmiş, profesyonel bir depo yönetim çözümüdür.

## 🚀 Projenin Evrimi (Evolution)

Projenin ilk sürümü (Prototip), standart Windows Forms bileşenleri ve temel bir kod yapısı kullanıyordu. Güncel sürümde ise proje tamamen yeniden yapılandırıldı.

### ✨ Frontend (Arayüz & UX) Güncellemeleri
Projenin görsel kimliği tamamen değiştirilerek kullanıcı deneyimi üst seviyeye taşındı:
- **Standart -> Modern:** Klasik Windows Forms görünümü yerine, **Guna2 UI** kütüphanesi ile modern bir dokunuş yapıldı.
- **Koyu Tema (Dark Mode):** Göz yormayan, estetik bir koyu tema odaklı tasarım uygulandı.
- **Neon Efektleri:** Aktif bileşenler ve gölgeler için marka rengine uygun neon parlamalar eklendi.
- **Hizalama & Düzen:** Tüm kontroller, matematiksel ortalama yöntemleriyle panel içine pixel-perfect hizalandı.

### 🏗️ Backend (Teknik Altyapı) Güçlendirmeleri
Projenin teknik borçları (technical debt) temizlendi ve kod yapısı sürdürülebilir hale getirildi:
- **Katmanlı Mimari (Layered Architecture):** Spagetti kod yapısı terk edilerek; Veri Erişimi (DAL), İş Mantığı (BLL) ve Sunum (UI) katmanlarına geçiş yapıldı.
- **Güvenlik:** Şifreler artık düz metin yerine, modern hashing algoritmaları ile saklanıyor.
- **SQLite Optimizasyonu:** Veritabanı bağlantı yönetimi iyileştirildi, veritabanı dosyası yoksa otomatik oluşturulma mantığı eklendi.
- **Hata Yönetimi:** Kullanıcıya yönelik detaylı ve görsel hata bildirimleri entegre edildi.
